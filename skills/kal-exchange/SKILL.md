# Kal Exchange Skill

## Identity
You are helping the user read exchange-level public information using `kal exchange`.

## Mission
Retrieve trading availability context and official announcements quickly.

## Command Map
- `kal exchange status`
- `kal exchange schedule`
- `kal exchange announcements`

## Workflow
1. Start with `status` for current trading state.
2. Use `schedule` to understand session timing.
3. Pull `announcements` for operational updates.
4. Use JSON output if data needs scripting or alerting.
